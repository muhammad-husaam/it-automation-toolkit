# 🧰 IT Automation Toolkit

> A PowerShell-based automation toolkit for IT support tasks, system diagnostics, and reporting.

## 📌 Overview
The **IT Automation Toolkit** is a PowerShell-based project designed to automate common IT support and system administration tasks.  
It reduces manual effort and improves efficiency by providing a single menu-driven interface for multiple system utilities.

---

## ⚙️ Features

✨ Network Diagnostics  
💻 System Information Report  
💾 Disk Space Monitoring  
🔧 Service Status Checker  
👥 User Account Audit  
🩺 System Health Check  
📦 File Backup Automation  
🌐 Network Auto-Fix Tool  
📊 HTML Report Generator

---

## 📄 HTML Report Includes

The generated report contains:

- 👤 Author name: **Muhammad Husaam**
- 🕒 Timestamp of execution
- 🖥 System information (OS, architecture, hostname)
- 💾 Disk usage summary
- 🌐 Network configuration details
- 📊 Clean formatted HTML output
